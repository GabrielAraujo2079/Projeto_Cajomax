## 📂 Estrutura de Pastas

```
/Projeto_Cajomax
│
├── /public                ← FRONT (o navegador vê)
│   │
│   ├── index.html         ← entrada principal (fica fora das outras pages)
│   │
│   ├── /pages             ← outras telas
│   │   ├── login.html
│   │   └── dashboard.html
│   │
│   ├── /css               ← estilos
│   │   ├── global.css     ← reset + variáveis + tema
│   │   ├── layout.css     ← estrutura (header, container)
│   │   └── components.css ← botões, cards, inputs
│   │
│   ├── /js                ← scripts do front
│   │   ├── main.js        ← eventos e DOM
│   │   └── api.js         ← comunicação com Node (fetch)
│   │
│   └── /assets            ← arquivos estáticos (tudo que não é código pique imagem)
│       ├── /images
│       ├── /icons
│       └── /fonts
│
├── /src                   ← BACKEND (Node)
│
├── server.js              ← inicia o servidor
└── package.json
```
